## Сборка механики

### Порядок сборки ноги

```
Стуктура одной ноги:

    [Корпус] 
        │
    ┌───┴───┐
    │  Hip  │  <- Бедро (вращение вокруг вертикальной оси)
    └───┬───┘
        │
    ┌───┴───┐
    │ Thigh │  <- Бедро (вращение вперёд/назад)
    └───┬───┘
        │
    ┌───┴───┐
    │ Calf  │  <- Голень (вращение вверх/вниз)
    └───┬───┘
        │
    ┌───┴───┐
    │ Foot  │  <- Стопа
    └───────┘
```

1. **Собрать бедро (hip)**
   - Установить сервопривод в корпус бедра
   - Закрепить 4 винтами

2. **Собрать голень (thigh)**
   - Прикрепить бедро к верхней части голени
   - Установить сервопривод в нижнюю часть голени

3. **Собрать стопу (calf)**
   - Прикрепить голень к стопе через сервопривод

4. **Повторить для всех 4 ног**

## Электрические соединения

### Схема подключения

```
                    ┌─────────────────┐
    ┌───────────────┤  LiPo Battery   │
    │               │   7.4V / 11.1V  │
    │               └────────┬────────┘
    │                        │
    │               ┌────────┴────────┐
    │               │   DC-DC Buck    │
    │               │   7.4V → 5V     │
    │               └────────┬────────┘
    │                        │
    │     ┌──────────────────┼──────────────────┐
    │     │                  │                  │
    ▼     ▼                  ▼                  ▼
┌───────┐          ┌─────────────────┐    ┌─────────┐
│ BPi   │          │    PCA9685      │    │  IMU    │
│  5V   │◄─────────┤  VCC (питание)  │    │ (опц.)  │
│  GND  │◄─────────┤  GND            │    │         │
│ GPIO2 │◄────────►│  SDA (I2C)      │    │         │
│ GPIO3 │◄────────►│  SCL (I2C)      │    │         │
└───────┘          └────────┬────────┘    └─────────┘
                            │
         ┌──────────────────┼──────────────────┐
         │                  │                  │
    ┌────┴────┐       ┌────┴────┐       ┌────┴────┐
    │ Leg LF  │       │ Leg RF  │       │ Legs LR │
    │ 0,1,2   │       │ 3,4,5   │       │ 6,7,8   │
    └─────────┘       └─────────┘       └─────────┘

Legs RR: каналы 9,10,11
```

### Таблица подключения сервоприводов

| Канал PCA9685 | Расположение | Сустав | Ось |
|---------------|--------------|--------|-----|
| 0 | Левая передняя | Бедро | Yaw (горизонт.) |
| 1 | Левая передняя | Бедро | Pitch (вертик.) |
| 2 | Левая передняя | Голень | Pitch |
| 3 | Правая передняя | Бедро | Yaw |
| 4 | Правая передняя | Бедро | Pitch |
| 5 | Правая передняя | Голень | Pitch |
| 6 | Левая задняя | Бедро | Yaw |
| 7 | Левая задняя | Бедро | Pitch |
| 8 | Левая задняя | Голень | Pitch |
| 9 | Правая задняя | Бедро | Yaw |
| 10 | Правая задняя | Бедро | Pitch |
| 11 | Правая задняя | Голень | Pitch |

## Проверка сборки

### 1. Проверка I2C

```bash
# На Banana Pi
sudo apt install i2c-tools
sudo i2cdetect -y 1
```

Должно показать адрес `0x40` (PCA9685).

### 2. Тест сервоприводов

```bash
# Перейти в директорию проекта
cd ~/robot_dog_ws
source install/setup.bash

# Запуск теста
ros2 run dog_control test_servos
```

### 3. Проверка диапазонов

Перед первым запуском убедитесь, что:
- Все сервоприводы отвечают на команды
- Диапазоны движения корректны
- Нет механических блокировок

См. [CALIBRATION.md](CALIBRATION.md) для настройки.
